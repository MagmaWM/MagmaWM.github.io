Magma Window Manager

MagmaWM is a versatile and customizable Window Manager / Wayland Compositor, currently in development and actively seeking contributions from the community. Built with the Smithay library and programmed in Rust, MagmaWM along with it's RON config provides users with flexibility and customization options. MagmaWM is licensed under GPLv3, ensuring that it remains open-source and free for all to use and contribute to.
